# 
## 
Полезные команды для терминала в Manjaro Linux, которые помогут обновить систему и выполнять другие важные задачи.

1. Обновление системы\
`sudo pacman -Syu`
     - `-S` — синхронизация пакетов
     - `-y` — обновление баз данных
     - `-u` — обновление всех пакетов

    `sudo pacman -Syu --needed manjaro-system` # перед обновлением проверьте наличие новостей о системе


2. Установка пакетов\
`sudo pacman -S имя_пакета`


3. Удаление пакетов\
`sudo pacman -R имя_пакета`\
`sudo pacman -Rns имя_пакета` # для удаления вместе с зависимостями, которые больше не нужны


4. Поиск пакетов\
`pacman -Ss ключевое_слово`


5. Очистка кэша пакетов\
`sudo pacman -Sc`\
`udo pacman -R $(pacman -Qdtq)` # для удаления всех неиспользуемых пакетов


6. Просмотр информации о пакете\
`pacman -Si имя_пакета`\
`pacman -Qi имя_пакета` # для установленных пакетов

7. Проверка целостности системы\
`sudo pacman -Qk`


8. Работа с AUR (пользовательские репозитории).\
Для установки пакетов из AUR используйте `yay` или `pamac`:\
`yay -S имя_пакета`\
`pamac build имя_пакета`


9. Просмотр списка установленных пакетов\
`pacman -Q`\
`pacman -Qe > paketi.txt` # чтобы экспортировать список в файл


10. Просмотр логов системы\
`journalctl -xe`\
`journalctl --since today` # для просмотра логов за сегодня
